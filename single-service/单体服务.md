# 创建服务

- https://www.w3cschool.cn/gozero/gozero-uvzq3nm0.html

```shell
mkdir go-zero-demo-single-service

cd go-zero-demo-single-service

go mod init go-zero-demo-single-service

# new 一个 go-zero 项目
goctl api new great

go mod tidy



Done ...


$ tree greet
greet
├── etc                       // 静态配置文件
│   └── greet-api.yaml
├── greet.api
├── greet.go                  // main , 程序启动入口
└── internal                  // 单个服务内部文件, 其可见范围仅为当前服务
    ├── config                // 静态配置文件对应的结构体声明目录
    │   └── config.go
    ├── handler               // 可选, http 服务会有这一层做路由管理, handler 为固定后缀
    │   ├── greethandler.go
    │   └── routes.go
    ├── logic                 // 业务目录, 所有 业务编码 需要的编码文件都存放在这里, logic 为固定后缀
    │   └── greetlogic.go
    ├── svc                   // 依赖注入目录, 所有  logic 层需要用到的 依赖 都在这里进行显式注入
    │   └── servicecontext.go
    └── types                 // 结构体存放目录
        └── types.go
        
```


# 启动服务

```shell
cd great

go run greet.go -f etc/greet-api.yaml
```

# 测试服务

```go
// 默认监听 8880 端口, 可以在配置文件里面修改
 curl -i -X GET http://localhost:8880/from/you
```
